What is a program?

  • A program is a sequence of instructions that have been crafted to do something.
  • Learning a programming language is like learning a foreign language, except you’re communicating instructions to a computer.

Rules of Python 3

  • Syntax: the grammar of Python
  • Reserved words: the vocabulary of Python

Syntax (Grammar)

Your program will only run if it follows all the rules of Python syntax.

For example, Python uses indentation to specify blocks of code.

Correct syntax 
            if 10 > 0:
                print('Dog!')
            else:
                print('Cat!')
Wrong syntax 
            if 10 > 0:
            print('Dog!')
            else:
                    print('Cat!')

Lists, Sets, and Dictionaries

Data structures are used to organize and store data.

  • Lists are used to store an ordered sequence of values.
  • Sets are like lists, but they do not store duplicate entries, and they are unordered.
  • Dictionaries are used to story values associated with keys.

Loops

  • Computers are often used to automate repetitive tasks.
  • Python has two types of loops that can be used to execute instructions repeatedly: the for loop and the while loop.

For loops

Break and continue

  • The break statement is used to exit early from a for or while loop.
  • The continue statement is used to skip the rest of the loop body and jump back to the for or while statement.
